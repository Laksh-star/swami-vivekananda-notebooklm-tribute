## How to Use This Repo

**To replicate this notebook:**
1. Start with `/01-planning/notebook-design-guide.md`‚Äîunderstand the architecture
2. Run prompts from `/02-source-generation-prompts/` to create your source materials
3. Upload sources to NotebookLM
4. Apply personas and studio prompts from `/03-notebooklm-prompts/`

**To adapt for another subject:**
- Swap Vivekananda references for your chosen figure/topic
- Keep the structure‚Äîquotes, biography, comparisons, action plans, Q&A
- Adjust personas to fit your context
