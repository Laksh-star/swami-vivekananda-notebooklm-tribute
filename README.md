## How to Use This Repo

**To replicate this notebook:**
1. Start with `01-planning/vivekananda_jayanti_notebook_guide.md`—understand the full architecture
2. Follow the checklist in `01-planning/vivekananda_notebook_todo.md`
3. Run prompts from `02-source-generation-prompts/` to create your source materials
4. Upload sources to NotebookLM
5. Apply studio and chat prompts from `03-notebooklm-prompts/`

**To adapt for another subject:**
- Swap Vivekananda references for your chosen figure or topic
- Keep the structure: quotes, biography, comparisons, action plans, Q&A
- Adjust personas to fit your context

**Estimated time:** 4-6 hours across 2-3 focused sessions
